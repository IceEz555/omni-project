‡πÇ‡∏Ñ‡πâ‡∏î‡πÉ‡∏ô‡πÑ‡∏ü‡∏•‡πå `bridge.js` ‡∏ó‡∏≥‡∏á‡∏≤‡∏ô‡πÄ‡∏´‡∏°‡∏∑‡∏≠‡∏ô "‡∏•‡πà‡∏≤‡∏°" ‡∏Ñ‡∏£‡∏±‡∏ö ‡∏Ñ‡∏≠‡∏¢‡∏ü‡∏±‡∏á‡∏†‡∏≤‡∏©‡∏≤ Arduino ‡πÅ‡∏•‡πâ‡∏ß‡πÅ‡∏õ‡∏•‡πÄ‡∏õ‡πá‡∏ô‡∏†‡∏≤‡∏©‡∏≤ JSON ‡∏ó‡∏µ‡πà‡πÄ‡∏ß‡πá‡∏ö‡πÄ‡∏Ç‡πâ‡∏≤‡πÉ‡∏à ‡πÇ‡∏î‡∏¢‡∏°‡∏µ‡∏à‡∏∏‡∏î‡∏™‡∏≥‡∏Ñ‡∏±‡∏ç 3 ‡∏à‡∏∏‡∏î‡∏Ñ‡∏£‡∏±‡∏ö:

#### **‡∏à‡∏∏‡∏î‡∏ó‡∏µ‡πà 1: ‡∏Å‡∏≤‡∏£‡πÄ‡∏õ‡∏¥‡∏î‡∏õ‡∏£‡∏∞‡∏ï‡∏π‡∏ü‡∏±‡∏á (Open Serial Port)**

‡∏ï‡∏£‡∏á‡∏ô‡∏µ‡πâ‡∏Ñ‡∏∑‡∏≠‡∏Å‡∏≤‡∏£‡∏ö‡∏≠‡∏Å‡πÉ‡∏´‡πâ‡πÇ‡∏õ‡∏£‡πÅ‡∏Å‡∏£‡∏°‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏ü‡∏±‡∏á‡πÄ‡∏™‡∏µ‡∏¢‡∏á‡∏à‡∏≤‡∏Å‡∏™‡∏≤‡∏¢ USB (COM Port) ‡πÇ‡∏î‡∏¢‡∏ï‡∏±‡πâ‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß (Baud Rate) ‡πÉ‡∏´‡πâ‡∏ï‡∏£‡∏á‡∏Å‡∏±‡∏ö‡∏ó‡∏µ‡πà Arduino ‡∏™‡πà‡∏á‡∏°‡∏≤

```javascript
// ‡∏ö‡∏£‡∏£‡∏ó‡∏±‡∏î 18-19: ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏Å‡∏≤‡∏£‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏ï‡πà‡∏≠

// ‡∏™‡∏≥‡∏Ñ‡∏±‡∏ç‡∏°‡∏≤‡∏Å: ‡πÉ‡∏ä‡πâ ReadlineParser ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏ö‡∏≠‡∏Å‡∏ß‡πà‡∏≤ "1 ‡∏Ç‡πâ‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏° = 1 ‡∏ö‡∏£‡∏£‡∏ó‡∏±‡∏î (\n)"
```

#### **‡∏à‡∏∏‡∏î‡∏ó‡∏µ‡πà 2: ‡∏Å‡∏≤‡∏£‡∏£‡∏±‡∏ö‡πÅ‡∏•‡∏∞‡πÅ‡∏Å‡∏∞‡∏Å‡∏•‡πà‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏° (Receive & Parse)**

‡πÄ‡∏°‡∏∑‡πà‡∏≠ Arduino ‡∏™‡πà‡∏á‡∏Ç‡πâ‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏°‡∏≤ (‡πÄ‡∏ä‡πà‡∏ô `deviceID|profileID|{"temp": 25}`) ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô `parser.on('data')` ‡∏à‡∏∞‡∏ó‡∏≥‡∏á‡∏≤‡∏ô

```javascript
// ‡∏ö‡∏£‡∏£‡∏ó‡∏±‡∏î 39: ‡πÄ‡∏°‡∏∑‡πà‡∏≠‡∏°‡∏µ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡πÄ‡∏Ç‡πâ‡∏≤‡∏°‡∏≤
parser.on('data', (data) => {
    // 1. ‡∏•‡πâ‡∏≤‡∏á‡∏Ç‡∏¢‡∏∞: ‡∏ï‡∏±‡∏î‡∏ä‡πà‡∏≠‡∏á‡∏ß‡πà‡∏≤‡∏á‡∏´‡∏ô‡πâ‡∏≤‡∏´‡∏•‡∏±‡∏á‡πÅ‡∏•‡∏∞‡∏ï‡∏±‡∏ß‡∏≠‡∏±‡∏Å‡∏©‡∏£‡πÅ‡∏õ‡∏•‡∏Å‡∏õ‡∏•‡∏≠‡∏°‡∏≠‡∏≠‡∏Å
    const cleanData = data.toString().trim();

    // --- ‡∏Å‡∏£‡∏ì‡∏µ‡πÄ‡∏õ‡πá‡∏ô‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏ó‡∏±‡πà‡∏ß‡πÑ‡∏õ (Standard Parsing) ---
    // ‡∏ö‡∏£‡∏£‡∏ó‡∏±‡∏î 83: ‡πÉ‡∏ä‡πâ‡∏ï‡∏±‡∏ß‡∏Ñ‡∏±‡πà‡∏ô "|" ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡πÅ‡∏¢‡∏Å‡∏ä‡∏¥‡πâ‡∏ô‡∏™‡πà‡∏ß‡∏ô‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•
    const parts = cleanData.split('|');

    // ‡∏™‡∏°‡∏°‡∏ï‡∏¥‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏Ñ‡∏∑‡∏≠: "sensor01|temp_v1|{\"val\": 30}"

    // parts[1] = "temp_v1" (Profile ID)
    // parts[2] = "{\"val\": 30}" (JSON Data)

    const deviceId = parts[0];
    const profileId = parts[1];
    const rawJson = parts.slice(2).join('|'); // ‡πÄ‡∏≠‡∏≤‡∏™‡πà‡∏ß‡∏ô‡∏ó‡∏µ‡πà‡πÄ‡∏´‡∏•‡∏∑‡∏≠‡∏°‡∏≤‡∏£‡∏ß‡∏°‡πÄ‡∏õ‡πá‡∏ô Data

    // ‡∏ö‡∏£‡∏£‡∏ó‡∏±‡∏î 92: ‡πÅ‡∏õ‡∏•‡∏á‡∏Ç‡πâ‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏° JSON ‡πÉ‡∏´‡πâ‡πÄ‡∏õ‡πá‡∏ô Object ‡∏à‡∏£‡∏¥‡∏á‡πÜ ‡∏ó‡∏µ‡πà JS ‡πÉ‡∏ä‡πâ‡∏á‡∏≤‡∏ô‡πÑ‡∏î‡πâ
    const metrics = JSON.parse(rawJson);
```

#### **‡∏à‡∏∏‡∏î‡∏ó‡∏µ‡πà 3: ‡∏Å‡∏≤‡∏£‡∏™‡πà‡∏á‡∏ï‡πà‡∏≠ (Publish to MQTT)**

‡∏û‡∏≠‡πÑ‡∏î‡πâ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏ó‡∏µ‡πà‡πÄ‡∏õ‡πá‡∏ô Object ‡∏™‡∏ß‡∏¢‡∏á‡∏≤‡∏°‡πÅ‡∏•‡πâ‡∏ß ‡∏Å‡πá‡∏™‡πà‡∏á‡∏Ç‡∏∂‡πâ‡∏ô‡πÑ‡∏õ‡∏£‡∏©‡∏ì‡∏µ‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á (MQTT) ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡πÉ‡∏´‡πâ Backend ‡∏ô‡∏≥‡πÑ‡∏õ‡πÉ‡∏ä‡πâ‡∏ï‡πà‡∏≠

```javascript

  ...metrics, // ‡πÅ‡∏ï‡∏Å‡πÑ‡∏™‡πâ data ‡∏Ç‡∏≠‡∏á sensor ‡∏≠‡∏≠‡∏Å‡∏°‡∏≤‡πÉ‡∏™‡πà‡∏ï‡∏£‡∏á‡∏ô‡∏µ‡πâ
};
```
